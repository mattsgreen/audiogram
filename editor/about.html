Audiogram

Audiogram is a tool for generating shareable videos from audio clips.

It was originally developed and shared by the American public broadcaster WNYC. It takes an audio file and converts it to a video file which can then be uploaded and shared on social media platforms. The video file is created by analysing your chosen audio file to create a moving waveform, which is then overlaid on a background image; the waveform acting as an visual clue that what the user sees also has sound.

BBC News Labs has taken WNYC's Audiogram tool and developed it further to make it easier for editorial staff to customise and access.
