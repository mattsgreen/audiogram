Welcome to Audiogram

Audiogram is a BBC News Labs prototype, based on original work by NPR's WNYC in New York, which converts audio into shareable video. Take a look at some examples on Twitter.

Access Required

We're currently testing this tool experimentally, and it is only available to whitelisted users.

If you would like to use Audiogram, please email Jonty.Usborne@bbc.co.uk requesting access.

We only restrict access for monitoring purposes - everyone is welcome to use the tool!
It is however only available on Reith, so only BBC staff email addresses can be whitelisted.

Please note:

This is a prototype. The tool and its infrastructure may present errors or be unavailable at times.
Please contact jonty.usborne@bbc.co.uk with error reports, as well as general feedback and feature requests.
